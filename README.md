# DSA_WebPage

🚀 Data Structures Visualizer
An interactive web application for learning and visualizing Data Structures through real-time simulations and animations. Users can explore Linear and Non-Linear Data Structures, perform operations, and understand their working in an intuitive way.

✨ Features
* 📚 Linear & Non-Linear Data Structures
* 🔍 Arrays, Linked Lists, Stacks, Queues, Trees, Graphs, Heaps, and Tries
* 🎮 Interactive simulators with real-time visualization
* 📊 Time & Space Complexity analysis
* 📝 Step-by-step operation explanations
* 🌙 Dark/Light mode support
* 🎨 Modern responsive UI with animations
* 📱 Mobile, Tablet, and Desktop friendly

🛠️ Tech Stack
* HTML5
* CSS3
* JavaScript
